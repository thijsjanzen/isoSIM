Demonstrate isofemales

ISOSIM

The package isoSIM was designed to simulate the generation of iso-female lines, and simulate genomic changes during, and after, formation of iso-female lines. Furthermore, it allows for simulation of effects after crossing individuals from iso-female lines.

library(isoSIM)
library(ggplot2)
packageVersion("isoSIM")
## [1] '0.46'

We will analyze whether we can genetically tell apart two populations, where one population was founded by crossing two isofemale lines that were created from the same source population, and the other population was founded by crossing two isofemale lines created from different source populations.

First, we generate two source populations using and then increase the founder labels in the second population to ensure that both populations do not share founder labels.

pop_1 <- isoSIM::create_population(pop_size = 100, number_of_founders = 4, total_runtime = 1000, morgan = 1, seed = 42)
## 0--------25--------50--------75--------100
## *****************************************
pop_2 <- isoSIM::create_population(pop_size = 100, number_of_founders = 4, total_runtime = 1000, morgan = 1, seed = 666)
## 0--------25--------50--------75--------100
## *****************************************
pop_2 <- isoSIM::increase_ancestor(pop_2, increment = 10)

To ensure that our two wild populations are not inbred, we estimate the average Linkage Disequilibrium (LD) for both populations using the function calculate_LD. In the absence of inbreeding, we expect the average LD to be low or zero.

mean(isoSIM::calculate_LD(pop = pop_1, sampled_individuals = 10, number_of_markers = 30, random_markers = TRUE)$LD_matrix, na.rm=TRUE)
## [1] 0
mean(isoSIM::calculate_LD(pop = pop_2, sampled_individuals = 10, number_of_markers = 30, random_markers = TRUE)$LD_matrix, na.rm=TRUE)
## [1] 0

Now we can start to generate isofemale lines from both populations, using .

isoFemales_pop_1 <- isoSIM::create_iso_female(pop_1, n=2, inbreeding_pop_size = 100, seed = 44)
## 0--------25--------50--------75--------100
## *************
##  After 568 generations, the population has become completely homozygous and fixed
## 0--------25--------50--------75--------100
## *********
##  After 355 generations, the population has become completely homozygous and fixed
isoFemales_pop_2 <- isoSIM::create_iso_female(pop_2, n=2, inbreeding_pop_size = 100, seed = 45)
## 0--------25--------50--------75--------100
## *************************
##  After 1180 generations, the population has become completely homozygous and fixed
## 0--------25--------50--------75--------100

##  After 298 generations, the population has become completely homozygous and fixed

Using the function we then create three populations, two where the isofemales are drawn from the same source population, and one where we mix the two, e.g. 1x1, 1x2 and 2x2, where isofemales 1 and 2 indicate isofemales from source populations 1 and 2 respectively.

pop_1_1 <- isoSIM::create_population_from_individuals(isoFemales_pop_1, pop_size = 1000, total_runtime = 1000, morgan = 1, seed = 5)
## 0--------25--------50--------75--------100
## *****************************************
pop_1_2 <- isoSIM::create_population_from_individuals( list(isoFemales_pop_1[[1]], isoFemales_pop_2[[1]]), pop_size = 1000, total_runtime = 1000, morgan = 1, seed = 6)
## 0--------25--------50--------75--------100
## *****************************************
pop_2_2 <- isoSIM::create_population_from_individuals(isoFemales_pop_2, pop_size = 1000, total_runtime = 1000, morgan = 1, seed = 7)
## 0--------25--------50--------75--------100
## *****************************************

Then, using the function we calculate all pairwise FST values. Here, we expect to find the highest FST for the 1x2 comparison.

f1 <- calculate_fst(pop_1_1, pop_1_2, sampled_individuals = 10, number_of_markers = 100)
f2 <- calculate_fst(pop_1_1, pop_2_2, sampled_individuals = 10, number_of_markers = 100)
f3 <- calculate_fst(pop_1_2, pop_2_2, sampled_individuals = 10, number_of_markers = 100)
f1
## [1] 0.6895556
f2
## [1] 0.9983333
f3
## [1] 0.6936819

Which confirms our hypothesis and demonstrates how combining isofemale lines from different source populations generates a population with maximal genetic diversity.
